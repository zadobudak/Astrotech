This is simple control and test library for Metu Comet team. 
Created for fixed wing UAVs. 
It is based on [PX4](px4.io) and [ROS](ros.org) and [MAVROS](mavlink.org/mavros/).

Still under development. 

To run the code have Ros and Mavros installed and working properly.
Then run the following commands:

```catkin build```

```source devel/setup.bash```
